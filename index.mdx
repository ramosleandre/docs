## What is OpenHosta?

OpenHosta is a powerful Python extension that seamlessly integrates Large Language Models (LLMs) into your development workflow, enabling **AI-powered function emulation** while maintaining native Python syntax and paradigms.

<Note>
**The future of development is human** - OpenHosta bridges the gap between human intent and code execution.
</Note>

## Quick Example

Write functions in natural language and let AI implement them:

```python
from OpenHosta import emulate

def translate(text: str, language: str) -> str:
    """
    This function translates the text into the specified language.
    """
    return emulate()

result = translate("Hello World!", "French")
print(result)
# Output: Bonjour le monde !
```

## Why OpenHosta?

<AccordionGroup>
  <Accordion title="Beyond Programming" icon="wand-magic-sparkles">
    Create complex functions that were previously impossible by defining semantic functions in human language. Handle language processing related to common sense or challenging parameters that are difficult to implement in native Python.
  </Accordion>

  <Accordion title="User-Friendly" icon="heart">
    Much simpler than LangChain or LlamaIndex. Direct access to prompts with the MetaPrompt class using Jinja2 templates. Print conversations and parsing steps to understand what happened.
  </Accordion>

  <Accordion title="Python Ecosystem" icon="python">
    Fully integrated into Python syntax. Uses docstrings and type annotations that Python developers already know. Compatible with Pydantic, typing module, and lambdas.
  </Accordion>

  <Accordion title="Reproducibility" icon="rotate">
    Make results reproducible by setting seeds for local Ollama installations. Essential for testing and debugging.
  </Accordion>

  <Accordion title="Open Source" icon="github">
    MIT licensed and community-driven. We believe in the sustainability and independence of the AI sphere. Count on your feedback and contributions.
  </Accordion>
</AccordionGroup>
